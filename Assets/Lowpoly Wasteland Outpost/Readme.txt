This package contains a large number of lowpoly modular parts, which will let you create various buildings, structures and obstacles for such levels as wasteland outposts, post apocalyptic towns and desert slums. It also includes different props, foliage, rocks, several ready-made buildings and obstacles, ground and grass textures and a demo scene.

You will find here over 125 prefabs:

- 9 ready-made buildings and 10 obstacles;
- 25 props;
- 4 ground materials;
- Plants, trees and rocks;

Can be used for  creating environment, perfect for mobile platforms.


This package has static lighting, but all lightmaps are deleted for the less package size. Try to rebake lighting if you want.
